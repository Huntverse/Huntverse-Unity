Kirist Development Tool - Documentation
========================================

This package contains comprehensive documentation for the Kirist Development Tool.

Files included:
- Kirist_User_Guide.pdf: Complete user guide with step-by-step instructions
- README.txt: This file

Quick Start:
1. Open Unity Editor
2. Go to Tools > Kirist > Open Kirist Window
3. Follow the user guide for detailed instructions

Features:
- Missing Script Finder
- Missing Material Finder  
- Missing Prefab Finder
- Prefab Analyzer
- Scene Analyzer
- Addressable Helper

For support and updates, visit the Unity Asset Store page.

Version: 1.0
Last Updated: 2024
